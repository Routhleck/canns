空间导航建模
============

场景描述
--------

空间导航是大脑最重要的功能之一。本系列教程将教你如何使用 CANN 模型和分层神经网络来模拟大脑的空间导航系统，包括：

- 网格细胞（Grid Cells）的多尺度表征
- 位置细胞（Place Cells）的空间编码
- Theta 节律在导航中的作用
- 路径积分和轨迹跟踪机制

你将学到
--------

1. 简单的路径积分机制
2. 分层神经网络的构建（条带细胞、网格细胞、位置细胞）
3. Theta 节律调制的实现
4. 复杂环境中的导航模拟
