CANN 动力学分析
================

场景描述
--------

连续吸引子神经网络（CANN）是一种强大的神经动力学模型，能够通过形成"bump"活动模式来表征连续的状态变量（如位置、方向等）。本系列教程将教你如何：

- 理解 CANN 的基本动力学原理
- 分析不同输入如何影响 bump 的形成和移动
- 可视化和分析 CANN 的响应特性
- 探索模型参数对动力学行为的影响

你将学到
--------

通过本系列教程，你将掌握：

1. **一维 CANN** 的设置和基本追踪任务
2. **二维 CANN** 的空间编码和动画可视化
3. **调谐曲线** 的计算和分析方法
4. **振荡性追踪** 的实现和应用场景

实际应用
--------

CANN 动力学分析在以下场景中特别有用：

- **工作记忆**：理解如何维持短期记忆状态
- **运动控制**：分析方向和速度的神经编码
- **感知决策**：研究刺激表征和注意力机制
- **空间导航**：作为后续更复杂导航模型的基础
