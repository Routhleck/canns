核心算法
--------

**BCM 规则**：

.. math::

   \Delta W = \eta \cdot \phi(y, \theta) \cdot x^T

其中：

- :math:`\phi(y, \theta) = y(y - \theta)` 是 BCM 调制函数
- :math:`\theta = \langle y^2 \rangle` 是滑动阈值

**关键特性**：

- **LTP 区域**（y > θ）：突触增强
- **LTD 区域**（y < θ）：突触减弱
- **自稳态**：阈值根据神经元活动自适应

- **发展神经科学**：理解关键期可塑性

生物学背景
----------

BCM 理论成功解释了：

- **关键期**：视觉经验对发展的时间依赖性
- **单眼剥夺**：眼优势柱的形成
- **对比度适应**：阈值的动态调整
- **选择性发展**：从非选择性到选择性的转变
