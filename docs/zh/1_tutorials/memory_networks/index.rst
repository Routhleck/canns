记忆网络训练
============

场景描述
--------

Hopfield 网络是最经典的联想记忆模型，能够存储和检索模式。本系列教程将教你如何：

- 使用 Hebbian 学习训练 Hopfield 网络
- 存储和恢复多种模式（包括图像）
- 分析网络的能量景观和存储容量
- 对比 Hebbian 和 Anti-Hebbian 学习的效果

你将学到
--------

1. Hopfield 网络的基本原理和训练
2. 一维二值模式的存储与检索
3. MNIST 手写数字的记忆任务
4. 能量函数分析和诊断工具
5. Hebbian 与 Anti-Hebbian 学习对比

适用人群
--------

- 对联想记忆感兴趣的研究人员
- 学习神经网络基础的学生
- 需要实现内容可寻址记忆的开发者

前置知识
--------

- 基本的神经网络概念
- 线性代数基础
- Python 和 NumPy

实际应用
--------

- **模式识别**：存储和识别视觉模式
- **错误修正**：通过联想记忆恢复损坏的数据
- **内容可寻址存储**：通过部分信息检索完整内容
- **认知建模**：理解人类记忆的神经基础

关键概念
--------

- **能量函数**：E = -∑ᵢⱼ Wᵢⱼ sᵢ sⱼ
- **吸引子**：模式作为网络动力学的稳定点
- **存储容量**：~0.14N（N 为神经元数）
