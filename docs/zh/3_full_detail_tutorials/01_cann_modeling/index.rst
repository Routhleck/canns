学习路径
--------

** 初学者：**

1. 从教程 1-4（基础部分）开始
2. 按顺序完成每个教程
3. 运行所有代码单元以理解概念
4. 进行参数实验

** 高级用户：**

- 教程 5：多尺度空间表示
- 教程 6：Theta 调制动力学
- 教程 7：具有位置细胞的复杂环境

先决条件
--------

- 对神经网络有基本了解
- 具有 Python 编程经验
- 熟悉 NumPy 和 matplotlib

预计时间
--------

- 基础教程（1-4）：约 2-3 小时
- 高级教程（5-7）：约 2.5 小时
- 总计：约 5 小时
